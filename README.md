# pitcms-test
[pitcms](https://pitcms.net/ )を使ってみる 

## [pitcmsを導入する](https://pitcms.net/docs/getting-started/installation)に従って進めてみる。
- pitcmsのダッシュボードでGitHubと連携する
  - [pitcmsのダッシュボード](https://app.pitcms.net/login?redirect=/projects)からgit accountでログイン
  - 言われるがままにgit経由でアクセス件付与。
- プロジェクトを作成する
  - プロジェクトが作成される
- プロジェクトのリポジトリにpitcms.jsoncファイルを作成、コミットし、GitHubにあげる
